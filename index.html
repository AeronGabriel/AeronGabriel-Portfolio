My Projects

Previous projects that I worked on

A Web-Based Violation and Incident Report System for Barangay bayan Luma I, Imus, Cavite

PHP | JavaScript | HTML | CSS

As part of our requirement, we developed a web-based system for Barangay Bayan Luma I in Imus, Cavite. It is an online data storage for all the reports such as incidents, violations, and blotters. We decided to develop this web-based system because we noticed that most of the barangays in the Philippines record their data through a log book which isn't an efficient way to store reports and it could be time consuming to find the old records. Our system is also capable of generating a barangay clearance if the person requesting has a clean record in their barangay.

Project Link: Violation and Incident Report System for Barangay Bayan Luma I
